# Text Highlight

A util hooks for search term highlighting

Usage:

`const ref = useTextHighlight(text: string, options?: MarkOptions, isHighlightEnabled: boolean)`

- In `useTextHighlight`, Pass `text` (the search term), `options` as the config object (docs can be found in [mark.js](https://markjs.io/))
and `isHighlightEnabled` in order to toggle the hook on/off (enabled as default).
- Pass `ref` to the component where you want to search for matches.

- In the example we are showing the most common options used while the `options` object is very rich and can be customized with a lot of abilities (please refer to [mark.js](https://markjs.io/) for more information)

````javascript
import { useTextHighlight, TextInput } from '@datorama/app-components';

const Component = () => {
  const [text, setText] = useState('');

  const ref = useTextHighlight(text);

  const handleChange = (event: ChangeEvent<HTMLInputElement>) => {
    setText(event.target.value);
  };

  return (
  	<div>
      <TextInput placeholder="Search text..." value={text} onChange={handleChange} />

			<div ref={ref}>

}
````
