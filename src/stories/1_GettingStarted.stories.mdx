# Getting Started

App-Components is a toolkit for developing Datorama Applications with ReactJS.
            

## Setup

Install the package in your project:

````shell script
// NPM
npm i --save @datorama/app-components

// Yarn
yarn add @datorama/app-components
````

Add the theme and custom color prefixes contexts (the latter is optional, but needed for branding svgs if the theme includes custom colors - see the [global styles page](/story/utils-global-styles--page?path=/story/utils-global-styles--page)):

````javascript
import { AppThemeProvider, lightTheme } from '@datorama/app-components';
import { ThemeProvider } from 'styled-components';

const App = () => (
  <AppThemeProvider theme={lightTheme} provider={ThemeProvider} customColorPrefixes={["c1", "c2", "c3"]}>
   { /* your app goes here*/ }
  </AppThemeProvider>
);
````

Add common styles:

````css
// add to the main css file
@import '~@datorama/app-components/index.css';
````
